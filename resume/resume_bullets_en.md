# English Resume Bullets

- Built an end-to-end KPI framework for a content platform, covering traffic, consumption quality, engagement, risk, and conversion with unified metric definitions.
- Implemented an anomaly monitoring pipeline (rolling baseline + deviation rules + segment alerts) and identified 24 actionable anomalies for faster issue detection.
- Led layered root-cause analysis across category, traffic source, user type, and time windows; pinpointed key drivers behind CTR decline, report-rate spikes, and CVR drops, with execution-ready recommendations.
- Evaluated a recommendation-card A/B test with core and guardrail metrics, quantifying trade-offs (CTR +7.94%, CVR +6.41%, Revenue +14.47%, but higher report rate), and proposed a staged rollout with risk controls.
